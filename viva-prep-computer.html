Definition of a Computer
An electronic machine capable of accepting data (input), processing it, storing it, and producing information (output) with the aid of programs. Data = raw facts. Information = processed, meaningful output.
input → process → outputdata vs information
✓
5 Generations of Computers
1st (1945–49): Vacuum tubes · 2nd (1950–60): Transistors · 3rd (late 1960s–70s): Integrated Circuits · 4th (1975–90): Microprocessors / PCs · 5th: AI & neural networks.
vacuum tubes → transistors → IC → microprocessor
✓
Types of Computers by Size
Supercomputer (fastest, weather/research) · Mainframe (large orgs, thousands of users) · Minicomputer (up to 200 users) · Microcomputer/PC (desktop, laptop, palmtop).
supercomputer = fastestminicomputer = 200 users
✓
CPU & Its 3 Components
CPU = Central Processing Unit = brain of computer. Speed in MHz/GHz. Three parts: MU (Memory Unit — fast access), CU (Control Unit — fetches & decodes instructions), ALU (Arithmetic Logical Unit — does all calculations).
MUCUALUMHz / GHz
✓
Memory & Storage Units
Bit (smallest) → Byte (8 bits, typical unit) → KB → MB → GB (1024 MB). RAM = temporary. ROM = permanent. Floppy 1.44 MB · CD 650–700 MB · DVD 4.7–8.5 GB · Blu-ray 25–50 GB.
RAM = temporaryROM = permanent1024 MB = 1 GB
✓
Input & Output Devices
Input: keyboard, mouse, scanner, touchscreen, light pen. Output: monitor (VDU), speaker, printer. Motherboard = main board connecting all peripherals. BIOS = Basic Input Output System.
keyboard = inputmonitor/printer = outputBIOS
✓
System vs Application Software
System software manages hardware (OS like Windows). Application software solves user tasks (MS Word, Excel, Photoshop). Computer cannot boot without an OS. GUI = Graphical User Interface. Icons = small pictures representing files/folders.
Windows = OS = system softwareMS Word = application
✓
Computer Virus & Anti-Virus
Virus: infectious program that corrupts data, deletes files, slows the computer, causes abnormal behaviour. Spreads via infected programs. Anti-virus detects and cleans known viruses. Examples: Avast, Norton, Kaspersky, Avira.
corrupts dataAvast / Nortoncold vs warm boot

MS Word Basics
Used for documentation. Extensions: .doc / .docx. Title bar default: "Document 1". Default font: 11pt. Default page: 8.5×11 inches. Default margins: 1 inch all sides. Gutter margin = binding side extra space.
.docx11pt default1 inch margins

Alignments & Line Spacing
4 alignments: Left (Ctrl+L) · Right (Ctrl+R) · Centre (Ctrl+E) · Justify (Ctrl+J). Line spacing: Single (Ctrl+1) · 1.5 (Ctrl+5) · Double (Ctrl+2). Word wrap = text moves to next line automatically.
Ctrl+L/R/E/JCtrl+1/2/5

Key Features

red = spellingwatermark5 change cases

Insert tab: pictures, shapes, SmartArt, charts, WordArt, symbols, page numbers. Review tab: spell check (F7), new comment, track changes. Superscript Ctrl+Shift+= · Subscript Ctrl+=. Columns for newspaper layouts.

Excel Basics
Used for calculations. Extensions: .xls / .xlsx. Title bar default: "Book 1". Columns = vertical (A, B, C…). Rows = horizontal (1, 2, 3…). Cells = intersection. Last column: XFD. Default sheets: Sheet1, Sheet2, Sheet3. Max sheets: 255.
.xlsxBook 1XFD = last column

Formulas & Operators
All formulas start with =. Operators: + (add) · - (subtract) · * (multiply) · / (divide) · ^ (exponent) · <> (not equal). Order of operations: Parentheses → Exponents → × ÷ → + −. Example: 5+2*7 = 19, not 49.
= starts all formulas* = multiply^ = power
✓
Key Functions
SUM(range) · AVERAGE(range) · MAX(range) · MIN(range) · COUNT(range — numbers only) · COUNTIF(range, criteria) · COUNTIFS (multiple criteria) · SUMIF · SUMIFS (first arg = sum range) · IF(condition, true, false) · AND (all true) · OR (any true) · MEDIAN · MODE · LARGE(range,n) · SMALL(range,n).
SUMIFS — sum range firstAND = all must be trueOR = any one true

MBS = ABS/12 · Transport 1% = (1/100)*MBS · Gross = MBS + House + TP + Feeding · Total Deductions = Tax + Dues · Net Pay = Gross − Total Deductions · Discount = TotalPrice*0.5 · Amount Paid = TotalPrice−Discount.

Internet & WWW
Internet = global network of interconnected computers. WWW = World Wide Web = system of interlinked web pages accessed via browser. URL = web address. HTTP = HyperText Transfer Protocol. ISP = Internet Service Provider (needed to connect). FTP = File Transfer Protocol (share files).
WWW ≠ InternetURL = addressHTTP / FTP

Web Browsers & Navigation
Browser = software to view web pages. Internet Explorer (Microsoft), Netscape Navigator (1994, first popular browser), Chrome, Firefox. Address Bar = type URL. Back/Forward = browse history. Homepage = default page on open. Favourites/Bookmarks = saved shortcuts. Hyperlink = click to navigate (cursor = hand).
IE = MicrosoftNetscape = 1994hyperlink = click to navigate
✓
Search Engines
Search engine = website that indexes web pages for keyword searching. Steps: open browser → go to www.google.com → type keywords → press Enter. Google = most used. Others: Yahoo, Bing, DuckDuckGo. Download = Internet → Computer. Upload = Computer → Internet.
Google = most usedkeywords → Enterdownload vs upload
✓
Email Basics
Email = Electronic Mail. Address format: username@domain.com (@ separates user from ISP). Email parts: To · CC (Carbon Copy — visible) · BCC (Blind Carbon Copy — hidden) · Subject · Body · Attachment. Client examples: MS Outlook, Mozilla Thunderbird. Webmail: Gmail, Yahoo Mail.
@ separates user/domainCC visible / BCC hidden
✓
Creating, Sending & Receiving
New email: Compose → To → Subject → Body → Attach → Send. Reply = responds to sender only. Reply All = sender + all CC. Forward = sends to new recipient. Inbox = incoming messages. Sent = copies of sent. Drafts = unfinished. Spam = junk/unsolicited — do not open attachments from unknown senders.
Reply vs Reply AllForward = new recipientSpam = junk
